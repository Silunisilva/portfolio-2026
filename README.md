# Portfolio (Vite + React + TypeScript + Tailwind)

This is a minimal starter portfolio single-page app with:
- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion animations
- CSS scroll-snap for section-by-section scrolling

Quick start:

```bash
cd path/to/Portfolio
npm install
npm run dev
```

Notes:
- The navbar highlights the active section using an IntersectionObserver.
- Sections animate into view with Framer Motion and the page uses scroll-snap for a smooth section-by-section feel.
